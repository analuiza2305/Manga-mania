## 📝 **Sobre o Projeto**

Bem-vindo ao *Manga Mania* — um projeto feito para todos os apaixonados por mangás!
Aqui você pode explorar, descobrir e se encantar com obras incríveis do universo mangá, tudo de forma prática e divertida.
